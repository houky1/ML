# SVM Realization Project

Реализация метода опорных векторов (Support Vector Machine) для задачи классификации.

## Описание

Был реализован метод опорных векторов без использования библиотеки scikit-learn для случаев с использование мягкого зазора и функции ядра.
Подробно описана математическая модель этих двух методов. Для наглядности показаны примеры обучения модели с визуализацией
и сравнением с использованием библиотеки scikit-learn.

## Установка и запуск

1. Клонировать репозиторий:
git clone https://github.com/houky1/ML.git
cd ML

2. Создать и активировать виртуальное окружение (рекомендуется):
conda create -n svm_env python=3.12
conda activate svm_env

3. Установить зависимости:
pip install -r requirements.txt

4.Запустить Jupyter Notebook:
jupyter notebook SVM_realization_project.ipynb

## Результат

Получилось создать обучающую модель на тестовом датасете, которая оказалась довольно точной по сравнению с scikit-learn. 
Конечно, результат может отличаться в зависимости от анализируемых данных. Проект выполнялся в учебных целях и создан для того, чтобы
понять, как происходит обучение модели в методе опорных векторов.
